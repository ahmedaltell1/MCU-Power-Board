# Custom MSP430 Embedded Systems PCB

## Overview

This project is a custom-designed embedded systems PCB developed in KiCad. The board integrates a regulated power delivery stage, an MSP430 microcontroller subsystem, sensor interfaces, switch inputs, and IoT communication support into a single embedded hardware platform.

The system was designed to demonstrate practical embedded hardware development, PCB design workflow, sensor integration, power regulation, and hardware/software interfacing.

Additionally, the project incorporates an ultrasonic water-level detection system capable of monitoring liquid levels through embedded sensing and real-time signal interfacing.

---

## Features

- Custom PCB designed in KiCad
- MSP430 microcontroller integration
- Regulated power delivery subsystem
- Ultrasonic water-level detection
- Sensor interface circuitry
- Embedded switch inputs
- IoT communication module support
- Modular embedded hardware architecture
- Manufacturable PCB layout and fabrication files
